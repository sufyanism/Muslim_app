# 📱 Zeba: Your Spiritual Journey

✨ **Zeba** is a beautifully crafted spiritual companion app built with **React Native**. It helps users stay spiritually connected by providing:
- 📍 Qibla Direction
- 🕰️ Prayer Times (location-aware)
- 📖 Access to the Quran

---
